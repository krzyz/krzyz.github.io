Detailed Description

Bloch functions calculated for a given lattice system.

BlochSystem class stores all the Bloch functions (or rather, the coefficients of their plane wave expansions) that are to be used in Wannier function calculations for a given system.

The number of the stored functions depends on the system size N, as the relevant quasimomentum wavewectors take the form , where are the reciprocal lattice basis vectors and , are sets of indices such that (in case of odd N) and (in case of even N). The indices actually used in this library (which are returned by mspace() helper function and stored in ms member) are actually , as only their differences (see RSystem) are used in the calculations. The functions are approximated as per cutoff parameter.
Construction of the BlochSystem object is done through the use of a builder:

auto bs = POWannier::BlochSystem::createWithPotential(V)
.setLaserWavelength(pi)
.setFourierCutoff(5)
.setSystemSize(20)
.setPotentialDepth(8)
.complete();

The call to createWithPotential() initializes the building process, while the call to complete() returns the final object. All the set calls are optional and if not used, the default values are used for corresponding members.
