Calculate and write to a file the values of a given function in the provided region in real space.

The function calculates values in the given range, which is given by two vectors (beg and end) in the lattice basis vectors ( ) space. The range is simply -dimensional cuboid (where is the dimension of the problem) in that space. In real space, the outermost edges are: and and the range is -dimensional parallelepiped.
