Detailed Description

template<class K, class V>
class POWannier::Cache< K, V >

Thread-safe cache that can be used to store the results of a given single argument function.

Usage:

Cache<int,double> cache([](int x) { return x/2.0; });
double y = cache.get(1); // y = 0.5

Kargument of the function.
