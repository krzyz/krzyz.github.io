Detailed Description

Wannier function.

Wannier functions are localized functions defined in the Bloch functions space (or rather, its subspace for a given set of bands ) in the system with periodic potential: . In this library, the coefficients are obtained by position operators diagonalization.

The creation of the Wannier is done through RSystem's getWannier() method.
